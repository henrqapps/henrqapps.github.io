Política de Privacidade

Última atualização: 2025

Esta Política de Privacidade descreve como o aplicativo [NOME DO SEU APP] coleta, usa e protege informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos descritos nesta política.

1. Informações Coletadas

O aplicativo não coleta, armazena ou compartilha dados pessoais identificáveis dos usuários. Nenhum dado como nome, e-mail, localização ou informações sensíveis é solicitado ou mantido.

2. Permissões Necessárias

O aplicativo pode solicitar permissões estritamente necessárias para seu funcionamento, como:

  • Câmera: utilizada apenas para leitura de QR Codes ou captura de imagens quando aplicável.
  • Armazenamento: utilizado apenas para salvar imagens geradas ou arquivos criados pelo próprio usuário.

Essas permissões não são usadas para coletar dados pessoais.

3. Uso de Serviços de Terceiros

O aplicativo pode utilizar APIs ou serviços externos para funcionalidades específicas. Nenhuma informação pessoal é enviada para esses serviços.

4. Segurança

Como o aplicativo não coleta dados pessoais, nenhum dado sensível é armazenado. Mesmo assim, todas as funcionalidades são desenvolvidas seguindo boas práticas de segurança.

5. Alterações Nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. A versão mais recente estará sempre disponível nesta página.
